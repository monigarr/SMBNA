## Documentation Philosophy

This public documentation focuses on:

- **Reproducibility**: How to reproduce research results
- **Extensibility**: How to extend the framework safely
- **Understanding**: How the system works at a high level

This documentation intentionally omits:

- Deployment procedures
- Security architecture and threat models
- Operational configuration and tuning
- Internal implementation details
